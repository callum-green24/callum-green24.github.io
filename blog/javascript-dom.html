If HTML, CSS and JS were a guitar

HTML: would be the body and the neck. It is the structure that holds everything together.

CSS:would be the design of the guitar. Its shape its colours. CSS is the style and how things are laid out within the structure.

Javascript:would be the musician plucking the guitar stringsm, making them move and create music. Javascript is the interaction and functionality of your webpage.

The DOM (Document Object Model)

The DOM is the thing you can pull up when inspecting a webpage. The doms like book which you can jump around to any chapter you please and make edits. You could use it as a contents page finding specific sections of code you can then alter.

Control flow and Loops

Control flow:is the order of operations your computer will follow. Like putting together a piece of furniture or saying the alphabet you follow along step by step, A to Z

Loopsa loop is a task that repeats until all of the criteria placed on them have been met. Like a traffic light turning red, orange, green although that is a loop without an end.

Arrays, Functions and Objects

Arrays

Arrays are containers for multiple ordered elements which can be accessed by the number in which the sit in that array. Imagine those advent calenders you used to get at christmas with the chocolates on each day. The array is the calander and the elements are the chocolates. How you access the element in the array is through the number they are positioned (remember the first number is 0)
